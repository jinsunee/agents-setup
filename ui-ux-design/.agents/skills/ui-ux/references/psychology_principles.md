# Design Psychology & Visual Theory

## Key Psychological Laws

- **Hick’s Law:** Minimize choices to speed up decision-making. (Use for: Menus, Pricing tables).
- **Fitts’s Law:** Make important targets large and close. (Use for: CTAs, Mobile nav).
- **Jakob’s Law:** Users prefer familiar patterns. (Use for: Icons, Login flows).
- **Miller’s Law:** Chunk information into groups of 5-7. (Use for: Forms, Phone numbers).
- **Peak-End Rule:** Optimize the most intense moment and the final interaction.

## Visual Hierarchy Principles

1.  **Size:** Larger elements are perceived as more important.
2.  **Color:** Bright/Bold colors attract attention first.
3.  **Whitespace:** Surrounding an element with space emphasizes it.
4.  **Texture/Style:** Textured or elevated (shadow) elements stand out over flat ones.

## Color Theory Basics

- **60-30-10 Rule:** 60% Primary/Neutral, 30% Secondary/Support, 10% Accent/Action.
- **Emotional Impact:**
  - _Blue:_ Trust, Security (Tech, Finance).
  - _Red:_ Urgency, Passion, Error.
  - _Green:_ Success, Growth, Health.
  - _Black/White:_ Luxury, Minimalism.

## Typography Guidelines

- **Hierarchy:** H1 > H2 > H3 > Body > Caption. Distinct steps in size and weight.
- **Readability:** Line height (leading) should be 1.4-1.6x the font size for body text.
- **Line Length:** 45-75 characters per line for optimal reading comfort.
